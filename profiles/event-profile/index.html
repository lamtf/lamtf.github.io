# The Device Event Profile

All these Stereotypes are intended be applied to State Machines in order to indicate the start of a Sequence of Actions (or functions).

# Stereotypes

# The OnClick Stereotype

A Sequence of Actions will happen when a given Button is Clicked. This stereotype can be suppressed because all transitions from a button will have the same meaning.

# The OnChange Stereotype

A Sequence of Actions will happen after a TextBox have it's contents changed

# The OnKeyPress Stereotype

A Sequence of Actions will happen when A Key is pressed

  • Not applied to all all scenarios

# The OnTick Stereotype

Start a Sequence of Actions after some Time

# The OnResult Stereotype

Start a Sequence of Actions after Finishing some Activity
