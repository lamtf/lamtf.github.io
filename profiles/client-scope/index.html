# The Client Scope Profile

Defines the database element scope on client, hiding some elements as needed.

# Stereotypes

# NotForClient

The current table can't remain on client

# ClientConst

The current table can't remain on client, but it appears as a configuration value (constant)

  • filter -> A set of columns to be stored on the client side

# ClientSingleData

The current table will appear on the client side, but no forms or lists can be provided. A single row should be inserted

  • filter -> A set of columns to be stored on the client side

# ClientDataArray

The current table will appear on the client side, but no forms or lists can be provided. Multiple rows can be inserted ans selected

  • filter -> A set of columns to be stored on the client side

# ClientCantChange

The Client Application can't make updating on the database forms and lists are also not permitted
