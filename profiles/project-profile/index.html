# The Todo Stereotype

Meaning that the current element is not implemented yet

  • It may receive a kind of Date as parameter

# The Late Stereotype

Meaning that the implementation is somehow late (it it makes sense)

  • It may receive a kind of Date as parameter

# The NotTested Stereotype

Meaning that the current element is being programmed but not tested

  • It may receive a kind of Date as parameter

# The NotPassed Stereotype

Meaning that the current element is already tested but not pass to the test

  • It may receive a kind of Date as parameter

# The Passed Stereotype

Meaning that the current element already passed to all tests

  • It may receive a kind of Date as parameter
