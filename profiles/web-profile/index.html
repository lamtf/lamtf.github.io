# Stereotypes

# WService

Simple Call to a Web Service and Web Service Definition footnote:[The definition of the Web Service must be defined by it's respective State Machine and implemented in a Server Language such as PHP or JSP]

# Redirect Stereotype

Simple Web Page Redirection based on the Code Flow

# PrintingService Stereotype

Prints a Document based on a given Template

  • The Name of the current State is the name of the template which will be used by the printer
  • A View must be provided as an Argument

# MailService Stereotype

Sends an E-Mail based on a given Template

  • The Name of the current State is the name of the template which will be used by the Mail Service
  • A View must be provided as an Argument
